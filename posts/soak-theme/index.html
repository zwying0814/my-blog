前言

好久不见，朋友们！我是 Cuteen 系列主题的作者阿卷👋。Cuteen 五岁了，我也走到了硕士毕业季🎓。

掐指一算，Cuteen 从 2020 年正式推出到现在，已经悄悄走过了 5 个年头，也迭代了5个大版本。这 5 年里，不仅主题在跟着我的技术成长慢慢迭代，我自己也完成了一段重要的人生进阶——一路读到了硕士，如今研三刚开学，已经站在了毕业的门槛前。回头看才惊觉，时间真的像握不住的沙，走得比想象中快太多。

这几年，我的技术栈也跟着经历了不少“更新换代”。从最开始敲代码时接触的 PHP，到后来逐渐聚焦、最终专精于 Web 前端领域，每一步都走得很扎实。期间为了让 Cuteen 变得更好，我做过不少尝试：比如探索 Next.js、Nuxt.js 这类服务端渲染框架，想给主题带来更流畅的体验；甚至还凭着一股热情，自己“手搓”过块编辑器，希望能优化内容创作的手感。

不过兜兜转转，最后还是决定使用静态博客的架构，由于改动巨大，我给她起了一个新名字 —— Soak。关于“为什么选择静态站点而非动态站点”，我整理了一些自己的思考，打算单独写一篇文章详细聊聊两者的区别和适用场景，这里就先不展开啦～

主题简介

作为一款基于开源框架 Astro 构建的博客主题，Soak 从底层就自带 “高性能” 基因 —— 不仅搭载了 Solid.js、Tailwind CSS 等前沿技术栈，更在加载速度、页面流畅度上实现了对传统站点的超越，让你的博客既能 “颜值在线”，又能 “跑得飞快”。

无需复杂配置，Soak 原生支持 MDX 格式文章是一大亮点。它打破了传统 Markdown 的局限，能轻松将交互组件（如按钮、图表）融入文字内容，让你不再受限于 “纯文本表达”，无论是写技术教程还是生活随笔，都能玩出更多创意。

同时，Soak 也完整继承了 Cuteen 主题的实用功能，并在此基础上优化体验：

风格自由切换：无需改代码，简单配置就能在 3.x 版本的卡片风、4.x 版本的简洁风之间自由切换，轻松适配不同审美偏好。

硬核功能拉满：原生支持 LaTeX 公式渲染（适合理工科博主）、强大的语法高亮（尤其支持 diff 语法，写代码对比超直观），还有自动切换的黑暗模式、方便互访的友情链接功能，覆盖博客创作的核心需求，让你专注内容本身，无需为功能适配费心。
